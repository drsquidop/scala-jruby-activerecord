JSR 223 demo that uses a JRuby gem, which in turn uses ActiveRecord.
See model-gem.

Run tests
---------

::

  sbt test

Development
-----------

Generate Eclipse project: ``sbt eclipse``.

Generate IntelliJ project: ``sbt gen-idea``.
